# Discord Mini Games Bot

Mini games are one of the easiest ways to boost activity in a Discord community. They create opportunities for members to interact, compete, and have fun.

XponentBot combines several engagement systems into one platform, including:

* Word Scrambles
* Boss Raids
* Leveling
* Leaderboards
* Rank Cards

Instead of managing multiple bots, communities can use a single solution for engagement and progression.
