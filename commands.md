# XponentBot Commands

XponentBot provides a complete set of leveling, engagement, giveaway, and mini-game commands for Discord communities.

## General Commands

### /daily

Claim your daily XP reward and build your streak.

### /rank [user]

View your rank card or another member's progress.

### /leaderboard

Display the top members ranked by XP.

### /rewards

View all available role rewards.

### /help

Display all available commands and usage information.

## Voting Commands

### /vote claim

Claim your XP reward after voting for XponentBot.

### /vote setup reward

Configure XP rewards granted to members for voting.

## Leveling & XP Management

### /xp add user amount

Add XP to a user.

### /xp remove user amount

Remove XP from a user.

### /xp set-level user level

Set a user's level.

### /xp reset user

Reset all XP data for a user.

### /xp freeze

Freeze or unfreeze XP generation server-wide.

## Server Configuration

### /settings view

View current server settings.

### /settings xp-rate amount

Adjust XP earned per message.

### /settings cooldown seconds

Set cooldown time between XP gains.

### /settings announce-channel channel

Configure level-up announcement channels.

### /settings toggle feature

Enable or disable specific features.

### /setup-logs

Create a private XP audit log channel.

## MEE6 Migration

### /import mee6

Import your existing MEE6 leaderboard into XponentBot.

## Rumble Royale

## Giveaways
