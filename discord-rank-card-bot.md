# Discord Rank Card Bot

Rank cards allow members to showcase their progress within a server. Visual progression systems help increase participation and make achievements more rewarding.

XponentBot includes customizable rank cards that display:

* Current level
* XP progress
* Server rank
* User profile information

Server owners can create a more immersive experience while giving members a clear view of their progression.
